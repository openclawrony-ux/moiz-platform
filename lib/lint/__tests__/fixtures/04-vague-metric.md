# Mobile app PRD

## Problem

Existing competitors require a 5-step onboarding. Users abandon before seeing value.

## Target user

First-time users of meditation apps, ages 25–45.

## User stories

- As a new user, I want to start a guided session in under 30 seconds so that I feel benefit before doubt.
- As a returning user, I want my last session to resume so that I do not have to re-pick everything.
- As a busy user, I want a 90-second option so that I can fit it into a coffee break.

## Non-goals

- We are not adding social features in v0.
- We are not localising beyond English in v0.

## Success metric

Users will love the app and we will be successful in the marketplace.
